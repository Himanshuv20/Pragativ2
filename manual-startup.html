üîß Troubleshooting

If Backend Won't Start:

  • Check if Node.js is installed: node --version
  • Install dependencies: npm install in backend folder
  • Check if port 5000 is free: netstat -ano | findstr :5000

If Frontend Won't Start:

  • Install dependencies: npm install in frontend folder
  • Check if port 3000 is free: netstat -ano | findstr :3000
  • Try: npm run build then npm start

If Chatbot Doesn't Respond:

  • Ensure backend server is running on port 5000
  • Check browser console for errors (F12)
  • Test direct API: http://localhost:5000/api/chatbot/intents
